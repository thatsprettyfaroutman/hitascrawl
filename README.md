Crawls hitas site for new apartments. Adds to list (`found.txt`) if new one that meets requirements is found.

### Usage

* `clone repo`
* `npm install`
* Modify `settings.json`
* `node crawler.js`
* Outputs `found.txt` which contains all found apartments
